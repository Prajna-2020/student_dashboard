Introduction

A **Student Portal** built using **Django** is a web-based application that enables students to efficiently manage their academic activities in a centralized system. Leveraging Django’s robust framework, the portal provides secure authentication, user-friendly interfaces, and seamless integration of essential features such as course registration, assignment submissions, exam schedules, and grade tracking. With Django’s powerful ORM (Object-Relational Mapping), the system efficiently handles database operations, ensuring smooth access to student records and academic data. Additionally, the portal supports role-based access control, allowing students, teachers, and administrators to interact with relevant modules. By utilizing Django’s scalability and security features, the Student Portal enhances communication, automates administrative tasks, and provides a structured digital environment for academic management.

Installation

You need to install package files/dependencies for this project to run. To do this, you must have Python and pip installed on your computer.

You can download and install Python from here. Pip is included with Python, so no separate installation is needed.

Once Python is installed, navigate to the root folder of the project and run the following command to create a virtual environment: 

python -m venv venv

After creating the virtual environment, activate it using the following command:

For Windows:

venv\Scripts\activate

For macOS/Linux:

source venv/bin/activate

Next, install the required dependencies using: 

pip install -r requirements.txt

This will install all necessary packages, including Django and other dependencies.

Now, apply the database migrations by running: 

python manage.py migrate

Once the migrations are complete, you can create a superuser for the Django admin panel (optional but recommended): 

python manage.py createsuperuser

Finally, to start the development server, use the command: 

python manage.py runserver

This will start the Django development server. Open your browser and visit http://127.0.0.1:8000/ to access the Student Portal.

Make sure to run all commands from the project's root directory where manage.py is located.
